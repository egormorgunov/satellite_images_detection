# Детектирование объектов на спутниковых изображениях
Программа по обнаружению объектов на спутниковых изображениях с использованием характерных **признаков Хаара**. На вход подаются снимки с Google карт, на которых присутствуют дома и другие прямоугольные здания. С помощью признака Хаара программа обрабатывает все области изображения и находит дом, после чего выделяет обнаруженный объект прямоугольной рамкой и сохраняет его как новое изображение.
